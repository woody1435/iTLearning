## 添加jdk源码  
1.找到%jdk_home%\jdkx.y.z\src.zip（本人jdk版本1.8.0_161）  
2.解压后放入自己项目中  
    此时debug调试可跟踪进源码中  
3.实现源码中添加注释  
    点击idea project structure -> SDKs -> 选中对应版本jdk -> Sourcepath选项卡 -> 移除掉原来的src.zip -> 将解压后引入项目的源码路径添加进来 -> 保存即可
    ![](./resources/jdk源码-编辑.jpg)
